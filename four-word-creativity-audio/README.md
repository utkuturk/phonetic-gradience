# four-word-creativity-audio

PCIbex template for creative sentence production with 4 prompts per trial.

- Prompt format: 4 words total (with one spoken prompt)
- Critical items: phonetic ambiguity (`ax~acts`, `clause~claws`)
- Non-critical items: clear controls + fillers
- Demo: https://farm.pcibex.net/r/kneWUO/

## Files

- `main.js`: experiment logic
- `practice.csv`: practice stimuli
- `items.csv`: main stimuli
- `audio_manifest.csv`: batch manifest for ElevenLabs generation
- `audio/`: output folder for MP3 files
